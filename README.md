# JTFilePicker
##一个超轻量级，简单好用的多文件选择器。

![image](https://github.com/william198824/JTFilePicker/blob/master/jtfilepicker.gif?raw=true)
###功能：

1.支持自定义文件图标

2.以字符串集合返回

3.超级轻量，超级简单

4.基于kotlin+databinding

###安装：

在你的moudle里添加如下代码：
```
dependencies {
    ......
    implementation 'com.william:JTFilePicker:1.0.0'
}
```
